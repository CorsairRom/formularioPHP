
Pasos para la instalación del proyecto PHP:

1. Utilizando XAMPP, inicie la creación del proyecto en la carpeta htdocs. 
Cree una nueva carpeta llamada 'formularioPHP', la ruta completa sería 'xampp\htdocs\formularioPHP'.

2. Clone los archivos del repositorio de GitHub (https://github.com/CorsairRom/formularioPHP.git) 
en la carpeta recién creada. Asegúrese de activar el servidor web y la base de datos de XAMPP.

3. Cargue la base de datos utilizando MySQL. Cree una nueva base de datos con el nombre 'formulariophp'.

4. En la carpeta 'database', encontrará un script SQL que le permitirá cargar las tablas 
y los datos en la base de datos recién creada. Ejecute este script mediante comandos SQL.

5. Con el servidor en ejecución, abra su navegador web y acceda a la siguiente ruta: 'http://localhost/formulariophp/' 
para visualizar el formulario.
    
    PHP Version 8.1.6
    10.4.24-MariaDB
